Olá, esse projeto foi desenvolvido com a finalidade aprimorar meus estudos no desenvolvimento backend.

Então foi posto em prática o uso da ferramenta ORM com SQLAlchemy para aprimorar meus conhecimentos e técnicas.

O banco de dados utilizado é o PostgreSQL integrado com o Docker, que veio no embalo para ter um pedacinho estudado
ao criar um container do PostgreSQL nele e se criar noções básicas sobre o Docker.

A ideia foi refatorar meu projeto CRUD com SQLite, dar uma enxutada nele e praticar esses novos recursos.

O projeto foi dividido em dois arquivos:
• main.py - Que é onde acontece a interação com o terminal, executando as ações do CRUD
• backend.py - Responsável pela conexão com o postgresql, criação da tabela e toda a lógica por trás do CRUD criado.
